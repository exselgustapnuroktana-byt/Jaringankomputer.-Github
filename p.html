Jaringan Komputer & Internet

I. Pendahuluan

A. Pengertian Jaringan Komputer

Jaringan komputer adalah kumpulan dari dua atau lebih perangkat komputasi (seperti komputer, server, printer, atau perangkat mobile) yang saling terhubung satu sama lain untuk tujuan berbagi sumber daya (seperti file, printer, dan koneksi internet) dan berkomunikasi.

B. Pengertian Internet

Internet (Interconnected Network) adalah sistem jaringan komputer global yang menghubungkan miliaran perangkat di seluruh dunia menggunakan standar protokol komunikasi, yaitu Transmission Control Protocol/Internet Protocol (TCP/IP). Internet adalah contoh paling besar dan kompleks dari sebuah Wide Area Network (WAN).

II. Konsep Dasar Jaringan

A. Jenis-jenis Jaringan Berdasarkan Area

Jenis Jaringan Cakupan Area Keterangan
LAN (Local Area Network) Area kecil (kantor, rumah, sekolah) Menghubungkan perangkat dalam area geografis terbatas.
MAN (Metropolitan Area Network) Area kota atau kampus besar Menghubungkan beberapa LAN dalam jarak menengah.
WAN (Wide Area Network) Area luas (antar kota, negara, benua) Menggunakan koneksi publik (seperti internet) atau privat yang jauh.
PAN (Personal Area Network) Area sangat kecil (sekitar 10 meter) Menghubungkan perangkat pribadi (misalnya, Bluetooth).

B. Topologi Jaringan

Topologi menggambarkan tata letak fisik atau logis dari perangkat dan kabel dalam suatu jaringan.

  • Topologi Bus: Semua perangkat terhubung ke satu kabel tunggal (backbone). Sederhana, tetapi sulit untuk mendeteksi kerusakan.
  • Topologi Star (Bintang): Semua perangkat terhubung ke satu titik pusat (Hub/Switch). Mudah dikelola, tetapi jika pusat rusak, seluruh jaringan lumpuh. [attachment_0](attachment)
  • Topologi Ring (Cincin): Setiap perangkat terhubung ke dua perangkat lainnya, membentuk jalur melingkar.
  • Topologi Mesh (Jala): Setiap perangkat terhubung ke setiap perangkat lain. Sangat andal (redundansi tinggi), tetapi mahal dan kompleks.

III. Model Jaringan (Model OSI dan TCP/IP)

Model jaringan mendefinisikan standar cara komunikasi data dilakukan antar perangkat.

A. Model OSI (Open Systems Interconnection)

Model konseptual yang terdiri dari tujuh lapisan (layer), mendefinisikan fungsi komunikasi jaringan secara terpisah. [attachment_1](attachment)

  1. Lapisan 7: Application (Aplikasi): Antarmuka pengguna (e.g., HTTP, SMTP).
  2. Lapisan 6: Presentation (Presentasi): Format data, enkripsi, kompresi.
  3. Lapisan 5: Session (Sesi): Mengelola sesi koneksi antar aplikasi.
  4. Lapisan 4: Transport (Transportasi): Segmentasi, koneksi *end-to-end* (TCP/UDP).
  5. Lapisan 3: Network (Jaringan): Pengalamatan logis (IP), *routing*.
  6. Lapisan 2: Data Link (Tautan Data): Pengalamatan fisik (MAC), kontrol akses media.
  7. Lapisan 1: Physical (Fisik): Media transmisi (kabel, sinyal, bit).

B. Model TCP/IP

Model yang digunakan secara praktis di internet, terdiri dari empat lapisan yang disederhanakan.

  • Lapisan Aplikasi: (Gabungan L7, L6, L5 OSI) Protokol aplikasi.
  • Lapisan Transport: (Sama dengan L4 OSI) TCP dan UDP.
  • Lapisan Internet: (Sama dengan L3 OSI) IP, ARP, ICMP.
  • Lapisan Akses Jaringan: (Gabungan L2, L1 OSI) Media fisik dan *framing*.

IV. Protokol Inti Internet

Protokol adalah seperangkat aturan yang mengatur bagaimana data ditransfer.

  1. IP (Internet Protocol): Bertanggung jawab untuk pengalamatan (*addressing*) dan *routing* paket data. Setiap perangkat di internet memiliki Alamat IP unik.
  2. TCP (Transmission Control Protocol): Protokol berorientasi koneksi yang menjamin pengiriman paket data yang andal dan berurutan.
  3. UDP (User Datagram Protocol): Protokol tanpa koneksi yang cepat dan efisien, cocok untuk data yang toleran terhadap kehilangan paket (misalnya, *streaming* video).
  4. HTTP (HyperText Transfer Protocol): Protokol dasar untuk transfer data di World Wide Web (WWW).
  5. DNS (Domain Name System): Menerjemahkan nama domain yang mudah diingat (misalnya, google.com) menjadi Alamat IP yang digunakan oleh komputer.

V. Perangkat Jaringan

Perangkat Fungsi Utama Lapisan OSI/TCP-IP
Hub Mengirimkan data ke semua port (tidak cerdas). Fisik (L1)
Switch Mengirimkan data ke port tujuan spesifik (cerdas). Tautan Data (L2)
Router Menghubungkan dua atau lebih jaringan berbeda dan menentukan jalur terbaik (routing). Jaringan (L3) / Internet
Modem Mengubah sinyal digital menjadi analog (modulasi) dan sebaliknya (demodulasi). Fisik (L1)

VI. Cara Kerja Internet dan World Wide Web (WWW)

A. Alamat IP dan Paket Data

Data yang dikirim melalui internet dipecah menjadi unit-unit kecil yang disebut paket. Setiap paket berisi:

  • Data yang sebenarnya.
  • Header (informasi pengirim, penerima, dan urutan paket).

B. Proses Routing

Ketika Anda mengirim data:

  1. Perangkat Anda memberikan paket tersebut ke Router lokal Anda.
  2. Router membaca Alamat IP tujuan pada header paket.
  3. Router menggunakan Tabel Routing untuk memutuskan router mana yang harus menerima paket selanjutnya.
  4. Proses ini berlanjut dari router ke router di seluruh dunia hingga paket mencapai tujuan akhirnya (server).

C. World Wide Web (WWW)

WWW adalah kumpulan dokumen yang saling terhubung dan sumber daya lain yang diakses melalui internet menggunakan HTTP.

Komponen Utama WWW:

  • Klien (Browser): Perangkat lunak (Chrome, Firefox) yang meminta dan menampilkan halaman web.
  • Server Web: Komputer yang menyimpan file web dan merespons permintaan klien.
  • URL (Uniform Resource Locator): Alamat unik untuk sumber daya di web (misalnya, https://www.google.com/).

VII. Keamanan Jaringan Dasar

Keamanan adalah aspek penting karena jaringan, terutama internet, sangat terbuka.

  • Firewall: Sistem keamanan jaringan yang memonitor dan mengontrol lalu lintas jaringan yang masuk dan keluar berdasarkan aturan keamanan yang telah ditetapkan.
  • Enkripsi: Proses mengubah data menjadi kode rahasia (*ciphertext*) untuk mencegah akses tidak sah. Data hanya dapat dibaca setelah didekripsi.
  • VPN (Virtual Private Network): Membuat koneksi aman dan terenkripsi di atas jaringan yang kurang aman (seperti internet), memungkinkan pengguna mengakses jaringan pribadi dari jarak jauh.
